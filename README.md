# Co Working – Responsive Shared Workspace & Business Services Landing Page HTML Template Built with Vite and Tailwind CSS

Co Working is a professional, modern, and high-conversion landing page template specifically crafted for shared office spaces, coworking environments, and rental agencies. It features a clean aesthetic, a powerful hero section with booking filters, and smooth interactive elements to showcase your facilities and build community trust.

Whether you are managing a **Dedicated Desk** hub, a **Private Office** facility, or a **Global Coworking Network**, this template provides a premium user experience to help you capture leads and fill your seats.

**Live Demo:** [https://coworking-template.netlify.app/](https://coworking-template.netlify.app/)

## 🛠 Getting Started

Follow these steps to run the template locally:

### 1. Install Required Packages

```bash
npm install
```

### 2. Start the development server

Start the local dev server by running:

```bash
npm run dev
```

This will open the project in your default browser at [`http://localhost:5173`](http://localhost:5173) (or another port if 5173 is busy).

### 3. Build for production (optional)

To create a production build, run:

```bash
npm run build
```

The compiled files will be generated in the `dist` folder.

## How to Customize

This template uses **Tailwind CSS** with a custom theme configuration inside `src/style.css`.

### Tailwind Configuration Highlights

- **Custom fonts** are defined using CSS variables like `--font-gilroy-bold`, `--font-gilroy-medium`, and others.
- **Brand colors** are defined as variables such as `--color-primary`, `--color-success`, `--color-dark`, and `--color-light`.
- **Responsive text scale** is applied for heading sizes across breakpoints using `@apply` in `src/style.css`.
- **Container** layout is centered and padded with responsive max width at larger breakpoints:
  - `px-4` horizontal padding
  - `mx-auto` centering
  - `lg:max-w-6xl` maximum width on large screens

### Key Styling Files

- `src/style.css` – Tailwind base import, custom theme variables, typography rules, layout utilities, and modal styling.
- `src/globals.css` – Currently empty in this project, reserved for global Tailwind utilities if needed.
- `src/main.js` – JavaScript entry for AOS, Lenis smooth scrolling, form behavior, modal interactions, and animated counters.

### Customize the page content

- `index.html` is the main landing page and contains the full template structure.
- Replace images inside `src/assets/` with your own visuals for a custom brand look.
- Update the hero text, section headings, and feature descriptions directly in `index.html`.
